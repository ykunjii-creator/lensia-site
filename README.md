# LENSIA LPTI Kiosk

LPTI(Lens Personality Type Indicator) 키오스크 콘셉트의 정적 웹 프로토타입입니다.

## 실행

1. 프로젝트 폴더를 VS Code로 엽니다.
2. `index.html`을 Live Server로 실행합니다.
3. 별도 패키지 설치는 필요하지 않습니다.

## 화면

- 홈: LPTI 키오스크 소개
- 테스트: Warm/Cool, Everyday/Unique, Puppy/Kitty, Large/Medium 4개 질문
- 결과: LPTI 코드, 성향 축, 홍채 분석, 추천 렌즈

## GitHub Pages

`index.html`이 루트에 있으므로 GitHub 레포의 `Settings → Pages → Deploy from a branch → main / root`로 배포할 수 있습니다.
