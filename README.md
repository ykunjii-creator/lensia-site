# LENSIA static prototype

업로드한 디자인을 기준으로 제작한 HTML/CSS/JS 프로토타입입니다.

## 실행 방법
1. 폴더를 VS Code로 엽니다.
2. `index.html`을 Live Server로 실행합니다.
3. 홈 화면의 CTA 버튼 또는 상단 `AI 분석` 메뉴를 누르면 분석 화면으로 이동합니다.

## 파일 구성
- `index.html`: 홈/분석 화면 마크업
- `style.css`: 전체 디자인 및 반응형 스타일
- `script.js`: 화면 전환과 버튼 이벤트
- `assets/`: 업로드한 시안에서 추출한 이미지 리소스

## 참고
- 별도 빌드 과정이 없는 순수 HTML/CSS/JS 버전입니다.
- 제품 구매 버튼은 데모 alert로 연결되어 있습니다.
- 실제 서비스에서는 분석 API, 카메라 업로드, 상품 DB 연결이 추가로 필요합니다.
